What now?

  1. Click Copy URL to clipboard above (or select the URL manually).
  2. Switch back to your terminal where npx claude-mcp auth --env=production --manual is waiting.
  3. Paste the URL and press Enter.
  4. The CLI will exchange the code for tokens and save them to your local config dir. Done.

No data on this page is sent to the maintainer. The OAuth code shown here is short-lived (~10 minutes) and useless on its own without your locally-stored client_secret. See the Privacy Policy for details.
